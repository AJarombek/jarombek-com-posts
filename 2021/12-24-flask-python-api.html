While researching API frameworks and libraries, I grew interested in the Flask framework. Flask is very lightweight, and allows engineers to quickly create APIs in Python without much opinionated tooling. For example, Flask does not come installed with a database access layer or ORM, allowing engineers to pick whichever database access library they prefer.
